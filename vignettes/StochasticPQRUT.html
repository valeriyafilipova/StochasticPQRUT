Vignette

This R package implements the Stochastic PQRUT model, which performs flood frequency analysis of simulated discharge values.The model simulates Precipitation, temperature and initial conditions and uses deterministic rainfall-runoff model to obtain the final discharge values. The return periods are estimated using plotting positions.

Workflow

  1. Load data. Example data is provided in this package

  2. Find critical duration

a=criticalduration(Q=Qd,P=P,qtT=0.9,PDFplots=FALSE,intEvent=7,writeResults=FALSE)

  1. Split the precipitation values into seasons and fit GPD or Exponential distribution to POT using the extRemes package
d=rainPOT(datarainfall=P,pathmain,durt=24,qFtset=0.94,qWtset=0.95,qSptset=0.98,qStset=0.94,distfunc="GP",
          writeResults=FALSE,PDFplots=FALSE)

h1=temprsim(incondFt=b$ntp1F,incondWt=b$ntp1W,incondSpt=b$ntp1Sp,incondSt=b$ntp1S,durt=24,tm=h1,writeResults=FALSE,PDFplots=FALSE)
